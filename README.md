基于 OpenDigger 的开源社区“协作引力”模型：开发者流动与项目演化动力学分析

通过引入物理学中的万有引力模型，利用 OpenDigger 提供的开源生态指标，量化开源项目间的开发者流动强度和生态吸引力。

1. 项目背景

在开源生态中，传统的指标（如 Star 数）难以反映项目间的动态关联。本项目提出“协作引力”模型，以 OpenRank 代表项目“质量”，以 开发者重合率 的倒数代表“协作距离”，从而探索 React 与 Vue 等主流框架之间的演化动力学。

2. 核心公式

协作引力F的计算逻辑遵循：

F=G（M1·M2）/d^2

其中：

M (质量)：由 OpenRank 指标定义，代表项目在全生态中的结构影响力。

d(协作距离)：定义为 1/Jaccard(P_1, P_2)。重合开发者越多，d 越小，引力 F 越强。

3. 项目结构

4. 运行方式

步骤 A：运行分析模型

执行核心脚本以完成数据清洗、建模计算并生成可视化趋势图：

python analysis_model.py


运行成功后，控制台会输出引力均值和最大值，并生成 gravity_model_results.csv 文件。

步骤 B：查看交互看板

直接在浏览器中打开 dashboard_v2.html 即可查看到基于真实数据构建的交互式数字大屏。

步骤 C：阅读分析报告

查看 final_project_report.md 获取完整的建模思路、2024年5月“引力奇点”的深度解读以及后期演进规划。

5. 核心发现 (2023-2024)

技术奇点：模型成功捕捉到 2024 年 5 月 React 19 发布引发的引力峰值 (F=56.42)。

生态耦合：React 与 Vue 已形成稳定的“双星系统”，开发者重合率维持在 5%-8% 区间，展现出极强的生态协同效应。
